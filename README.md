## 1. Jekyll  설치 확인

```sh
gem install jekyll bundler
```

### 2. 의존성 설치

```
bundle install
```

### 3. Gemfile.lock이 오래되었을 경우, 업데이트 하기

```
bundle update
```

### 4. 로컬 서버 실행

```
bundle exec jekyll serve
```
